display、visibility、opacity 比较和继承性探究

关于display、visibility、opacity三者的细节，和对比概览查看另一篇文章： CSS（四）视觉属性

视觉上的特征比较

  • display:none，不渲染元素
  • visibility:hidden，渲染元素，但是不绘制元素，元素所在位置会被占用
  • opacity:0，渲染元素，绘制元素，只是元素透明度为0，看不见而已

子元素的表现

  • display:none不存在继承性一说，因为父元素没有渲染，意味着所有子元素都不会被渲染
  • visibility和opacity都具有继承性
  • 父元素设置visibility:hidden，子元素可以通过设置visibility:visible重新显示出来
  • 父元素设置opacity:1，子元素无法通过改变opacity显示出来
