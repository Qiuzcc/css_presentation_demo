（2）grid-template-rows 定义每一行的高度

  • 备注：grid子元素没有设置宽高
  • 在没有设置grid-item的高度时，高度默认填满grid网格的高度
  • 因为div是块级元素，所以宽度占满父容器（即单个网格）的宽度
  • 如果不设置grid-template-rows属性，则grid的行高默认为子元素的高度
grid-template-rows： 不设置 100px 50px 100px 100px 150px 200px

  • 参数为2位时，行距在前，列距在后
  • 对grid容器边界没有影响

（4）place-items 网格中元素的位置，水平位置（justify-items）和垂直位置（align-items）的简写

  • 备注：网格的行高为250px，列宽为1fr，子元素的宽高为150px
  • 为了体现不同place-items的差异，需要将子元素的宽高设置的比网格的宽高值小
  • 子元素设置了宽高的情况下，默认值stretch将失效，子元素不会拉伸；此时取而代之的默认值是start
  • 参数为2位时，水平位置在前，垂直位置在后
place-items： start center end stretch（默认值） start end

（5）grid-template-areas 区域化网格

  • 使用方式：容器使用 grid-template-areas 声明网格区域，子元素使用 grid-area 指定占用的网格区域
  • 注意：（1）area名字需要填满所有网格
  • 注意：（2）同一个area名只能出现在连续的网格中
  • 注意：（3）连续网格必须组成一个正方形

（6）grid-auto-flow 容器布局流

  • 划分网格以后，容器的子元素会按照顺序，自动放置在每一个网格。取值范围如下：
  • （1）row（先行后列），指定列数，从左到右、从上到下填充网格元素，必要时自动增加新行
  • （2）column（先列后行），指定行数，从上到下、从左到右填充网格元素，必要时自动增加新列
  • （3）dense，先行后列，如果本行的剩余单元格放不下下一个元素，将试图用后面较小的子元素填补这个空白的空间

grid-auto-flow: row （需要指定列数，grid-template-columns: 1fr 1fr 1fr;）
